# Ethical Use Notice

## Purpose
This repository is provided as a governance-first technical exploration focused on
human-in-the-loop NLP systems, auditability, and ethical system design.

## Intended Use
This project is intended for:
- Research and educational purposes
- AI governance and safety analysis
- System design discussion and review

## Prohibited Use
The authors explicitly discourage and disclaim responsibility for any use of this
repository in:
- Autonomous decision-making without meaningful human oversight
- Surveillance, profiling, or behavioral manipulation of individuals
- High-risk domains involving psychological, legal, or physical harm
- Systems designed to evade accountability or human responsibility

## Human Authority
Any system derived from this work must preserve human authority as a non-negotiable
control layer. Outputs must remain advisory, not directive.

## No Warranty
This repository is provided "as is", without warranty of any kind. The authors assume
no liability for downstream use or misuse.

## Ethical Alignment
Use of this repository implies acknowledgment of these principles and an obligation
to uphold human dignity, accountability, and transparency.
